## Stack Tecnológico

Este proyecto utiliza las siguientes tecnologías principales:

- **Backend**: Django 4.2.7
- **Admin Interface**: django-material-admin 1.8.6
- **Generación de PDF**: django-weasyprint, weasyprint, reportlab, xhtml2pdf
- **Análisis de Datos**: pandas, numpy, matplotlib
- **Código QR**: qrcode
- **Seguridad**: cryptography, defusedxml
- **Otros**: Múltiples librerías de soporte (ver requirements.txt)

Para información completa sobre dependencias, consulta `requirements.txt`.
